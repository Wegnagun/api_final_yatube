# api_final
api final

### Установка (windows)
`python -m venv venv `

`venv/Scripts/activate `

`python -m pip install --upgrade pip `

`pip install -r requirements.txt `

### Запуск
Перейдити в дирректорию yatube_api и выполните миграции:

`python manage.py migrate `

Запустите сервер:

`python manage.py runserver`
